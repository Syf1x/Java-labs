# Система менеджмента сотрудников в офисе

Данный проект разработан в рамках учебной дисциплины **«Программирование на языках высокого уровня»**. Приложение представляет собой систему для автоматизации учета сотрудников, управления структурой офиса и отслеживания рабочих процессов.

---

## О проекте
Основная цель системы — упростить взаимодействие и работу с персоналом главам отделов, руководству и HR
